# mMetrics
Metrics service for collect statistics from your applications.

## Technologies
* SpringBoot 
* Vue.js
* MySQL
* Docker

## How to run
This application will be dockerized, so you need to have Docker installed on your machine. 
Soon you can use it... **Stay tuned!**
